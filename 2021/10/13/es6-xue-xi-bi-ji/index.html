ECMAscript

ECMAScript是一种由Ecma国际（前身为欧洲计算机制造商协会，European Computer Manufacturers Association）通过ECMA-262标准化的脚本程序设计语言。这种语言在万维网上应用广泛，它往往被称为JavaScript或JScript，所以它可以理解为是JavaScript的一个标准,但实际上后两者是ECMA-262标准的实现和扩展

ECMAScript第六版简称es6

为什么要学习es6

  • es6的版本变动内容最多，具有里程碑意义
  • es6加入许多的新的语法特性，编程实现更加简单，高效
  • es6是前端发展趋势，就业必备技能

        const banji={
            name:'终极一班',
            stus:['小明','小宁','小天','knight'],
            [Symbol.iterator](){
                //创建索引
                let index=0;
                let _this=this;
                return {
                    next:function(){
                        if(index<_this.stus.length){
                            const result={
                            value: _this.stus[index],
                            done:false
                            }
                            index++;
                            //返回结果
                            return result;
                        }else{
                            return {
                                value:undefined,
                                done:true
                            }
                        }
                        
                    }
                }
            }
        }
        //遍历这个对象

            console.log(v);//banji is not iterable
